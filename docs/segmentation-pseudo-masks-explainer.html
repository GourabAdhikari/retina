New plan section: self-generated segmentation

We create the masks ourselves.

RASTA has no human-drawn vessel, FAZ, or flow-deficit masks. So the project now treats segmentation as weak, automated mask generation: classical image processing first, optional frozen student model second, classifier fusion last.

Core rule.

These are pseudo-masks, not manual ground truth. They can guide the classifier and produce interpretable biomarkers, but papers and docs must not claim human-validated segmentation unless we later annotate a validation subset.

Sup / Deep

Vessel masks

CLAHE, vesselness filtering, adaptive thresholding, morphology cleanup, small-component removal.

Sup / Deep

FAZ masks

Find central avascular low-flow component, smooth contour, reject implausible area or off-center masks.

CC

Flow-deficit masks

Local thresholding finds dark hypoperfusion regions, then morphology and area filters remove noise.

Where the student fits.

The student model is upstream of the disease classifier. It is not the classifier. It learns to imitate the classical pseudo-masks and then produces standardized masks for every image.

1. Preprocess OCTANormalize, CLAHE, resize, quality-control image.
2. Classical pseudo-maskGenerate deterministic vessel, FAZ, and CC flow-deficit masks without using disease labels.
3. Optional studentTrain Attention U-Net or U-Net++-style model on pseudo-masks. Freeze it before classifier training.
4. BiomarkersExtract density, skeleton density, FAZ geometry, and flow-deficit burden. Normalize with training-fold stats.
5. Classifier fusionUse OCTA images + F_mask + biomarkers + clinical features.

Attention U-Net or U-Net++?

Both are allowed. The updated plan can keep Attention U-Net as the light default and evaluate U-Net++ when pseudo-masks look stable enough to justify the extra capacity.

ChoiceWhy use itRisk
Attention U-NetSimpler, lighter, easier inside notebook, less likely to overfit noisy pseudo-labels.May miss fine capillary detail.
U-Net++Better multi-scale refinement; attractive for thin OCTA vessels and FAZ boundaries.Can learn pseudo-label noise if QC is weak.
Default policyStart classical-only. Add student only if masks improve by visual/QC review. Freeze student during classifier training.Needs montage review and ablation: classical masks vs student masks.
Do not joint-train by default.

Classifier gradients can distort masks. End-to-end segmentation/classification is only an experimental variant after mask-quality monitoring exists.
